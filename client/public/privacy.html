Privacy Policy

Last updated: April 5, 2026

World Weaver Chronicles stores account information (username and password hash) and game progress saves.

Data We Process

  • Authentication data: username and password hash.
  • Game state data: progression, inventory, quests, economy state, and settings.
  • Basic technical logs for API reliability and security.

Purpose

  • To authenticate users.
  • To store and restore game progress.
  • To protect service security and stability.

Contact

For privacy requests, contact the project operator through the distribution channel where this app is provided.
